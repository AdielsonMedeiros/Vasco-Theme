## Sobre

Este tema foi desenvolvido para oferecer um ambiente de desenvolvimento visualmente agradável, utilizando o alto contraste e as cores históricas do Club de Regatas Vasco da Gama. Ideal para longas sessões de programação com conforto visual e identidade.

## Funcionalidades

### Cores
Um tema "Dark High Contrast" elegante com acentos em vermelho, branco e preto.
- Leitura confortável.
- Sintaxe destacada para fácil identificação.
- Interface limpa e minimalista.

### Ícones (Vasco Icon Theme)
Ícones personalizados substituindo os padrões do VS Code:
- **Linguagens**: Cruz de Malta (Sólida, Outline, Círculo) para HTML, CSS e JS.
- **Suporte**: Ícones dedicados para React, TypeScript, Python, Java, Docker, Git, entre outros.

### Snippets
Atalhos incluídos para agilizar o desenvolvimento:

| Prefixo | Descrição | Resultado |
| :--- | :--- | :--- |
| `vasco` | Insere a Cruz de Malta em ASCII Art | Bloco de comentário com arte |
| `console-vasco` | Log estilizado no console do navegador | `console.log` com estilo visual |
| `vasco-saudacao` | Comentário de saudação | `// O Sentimento Não Pode Parar! /+/` |

## Instalação

1. Abra o **VS Code**.
2. Acesse a aba de **Extensões** (`Ctrl+Shift+X`).
3. Pesquise por **"Vasco da Gama Theme"**.
4. Clique em **Instalar**.
5. Selecione o tema de cores **Vasco da Gama** e o tema de ícones **Vasco da Gama Icons**.
