## Instalação

1. Instale via VS Code Marketplace ou arquivo VSIX.
2. Selecione o tema: **Vasco da Gama**.

## Ícones Personalizados

O tema inclui ícones exclusivos:
*   **HTML**: Cruz de Malta Sólida
*   **CSS**: Cruz de Malta Outline
*   **Javascript**: Cruz no Círculo
*   E suporte para dezenas de outras linguagens!

## Snippets (Atalhos)

Tente usar:
*   `vasco-ascii`: Arte em código
*   `log-vasco`: Console.log estilizado
